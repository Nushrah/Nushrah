TriChat-Lite – A Minimal Multi-Transport Chat Toolkit

TriChat-Lite is a concise networking lab that proves mastery of core socket programming concepts. A shared length-prefixed protocol is implemented three times: an asyncio WebSocket server for modern browser-friendly messaging, a classic TCP chat server using Python threads, and a UDP broadcast chat enhanced with a tiny reliability shim. The project highlights protocol design, event-driven vs. blocking concurrency trade-offs, and cross-platform deployment, making it a standout yet approachable addition to any résumé.
